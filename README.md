# Task-3-Filtering-Data-Using-WHERE-AND-OR-LIKE
SQL queries file demonstrating advanced filtering

##  Objective
The objective of this task is to understand and implement SQL data filtering techniques using the `WHERE` clause and related operators. This task focuses on retrieving precise and meaningful data from a database by applying conditions, which is essential for real-world database applications.

---

##  Tools Used
 ** Tool:** MySQL Workbench    

---

## Concepts Covered
- Filtering records using `WHERE`
- Combining conditions using `AND` and `OR`
- Pattern matching using `LIKE`
- Using wildcards `%` and `_`
- Filtering multiple values using `IN`
- Range filtering using `BETWEEN`
- Handling `NULL` values using `IS NULL` and `IS NOT NULL`
- Improving query readability using table aliases
- Understanding SQL query execution order
